== Description ==



Display beautiful particles on your website :)

You can set particles quantity, their speed, their opacity and where to display them (everywhere or one specfic page).

And you can also randomize quantity and speed of the particles.

See demo of the plugin [here](https://www.info-d-74.com/en/light-particles-demo/)

And if you want more option there is a [Pro version of this plugin](https://www.info-d-74.com/en/produit/light-particles-pro-plugin-wordpress-2/)



== Installation ==



1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Go to Settings => Light Particles
